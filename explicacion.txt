Una vez aca adentro de nuestro editor de codigo vamos a abrir una nueva terminal, pueden abrirla en la opcion
del menu de arriba que dice "Terminal", sino tambien con los comando "control + J", esto es funcional solo con
windows

En nuestra terminal vamos a poner el siguiente comando
npm init -y
esto nos genera un archivo JSON,
donde podremos ya ir instalando nuestras dependencias que vamos a utilizar
Nosotros para este proyecto vamos a necesitar instalar las siguientes dependencias

express mongoose cors dotenv nodemon

Para instalarlas vamos a nuestra terminal nuevamente y ponemos npm install y el nombre de todas las dependencias
que necesitamos instalar

Esto nos generara ya nuestros node_modules y nuestro package_look.json, para ver de que estas se hayan instalando
correctamente vayamos a ver nuestro package.json, como vimos ahi estan todas instaladas correctamente

ahora vamos a hacer un nuevo archivo que va a hacer nuestro servidor, esto puede tener cualquier nombre pero es muy
comun verlo como index o server o algo similar, nosotros en este caso vamos a llamarlo Server.js

perfecto, ya tenemos nuestro archivo
server.js, ahora empecemos a
codear un poco...

como estamos trabajando con Node
vamos a estar trabajando con CommonJs, por lo 
tanto vamos a utilizar el "require" para 
referirnos a las dependencias que queremos 
utilizar o algun modulo que necesitamos en 
nuestro archivo server.json

empecemos requiriendo los dotenv


Que estamos haciendo ahi, estamos modificando
nuestros scripts para poder iniciar nuestro server
mas facilmente, en "start" estamos diciendo
que si corremos ese comando en nuestra terminal
va a correr node el server pero no va a escuchar 
los cambios, en cambio con "dev" lo hacemos con
nodemon, este cada ves que se realicen cambios
en nuestro archivo server.js va a reiniciar el 
servidor automaticamente para que se reflejen
dichos cambios :)
